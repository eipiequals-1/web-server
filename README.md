## Requirements
- GNU Make
- C++ 20

## Building
- Clone: ```git clone https://github.com/eipiequals-1/web-server```
- Build: ```make``` or ```make -j 8``` depending on the number of cores available.
- Run: ```./bin/web_server```

- View web-server/main.cpp for a workflow overview!
